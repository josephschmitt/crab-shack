# copy-editor

Takes an editorial pass on existing prose to improve clarity, conciseness, and flow while preserving the author's voice. Applies Strunk & White sensibilities: cuts AI tells (tricolons, staccato runs, em-dash overuse, marketing filler), tightens sentences, and fixes passive voice — without rewriting the piece or stripping its personality.

**Trigger:** "edit this", "tighten this up", "copy edit", "take an editorial pass", or paste text and ask if it sounds right.
