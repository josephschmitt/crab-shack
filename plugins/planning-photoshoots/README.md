# planning-photoshoots

A conversational photography companion for planning shoots. Asks about goals, checks weather and location intel, then recommends a gear loadout, style direction, and practical tips tailored to what's in your bag. Remembers your kit and skill level across sessions so you don't have to re-list every lens.

**Trigger:** "what lens should I bring", "how should I shoot this", "photo tips for X", "planning a photo walk", or casual mentions like "I want to get some good shots while we're there."

## Install

### Claude Code

```
/plugin marketplace add josephschmitt/crab-shack
/plugin install planning-photoshoots@joes-crab-shack
```

### Claude.ai (web or desktop)

Download [`planning-photoshoots.zip`](https://github.com/josephschmitt/crab-shack/releases/latest/download/planning-photoshoots.zip) from the [latest release](https://github.com/josephschmitt/crab-shack/releases/latest), then in Claude click **+** → **Create skill** and upload the zip.
